## Install

run the   
`install.windows.ps1`

## Running tests

Execute the tests with:

```bash
$ ./run_tests.bat
```

## Rust Installation

Install Rust-up
https://www.rust-lang.org/learn/get-started

VSCode with plugin
Use the vs code plugin "rust-analyzer"


## Formatting Code

To format your solution, inside the solution directory use

```bash
cargo fmt
```
